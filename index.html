Computer Languages

HTML

Hypertext Markup Language is the standard markup language for documents designed to be displayed in a web browser. It allows the user to create and structure sections, paragraphs, headings, links, and blockquotes for web pages and applications.

CSS

Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language. The major characteristics of CSS include styling rules and handling events which are applied to various elements in your document.

Javascript

JavaScript is a dynamic computer programming language. It is lightweight and most commonly used as a part of web pages, whose implementations allow client-side script to interact with the user and make dynamic pages. It has object-oriented capabilities.

Python

Python is an interpreted, object-oriented, high-level programming language with dynamic semantics. Python's simple, easy to learn syntax emphasizes readability and reduces the cost of program maintenance. Python supports many modules and packages.

PHP

PHP is a popular general-pupose, server side scripting language, that is used to develop Static websites or Dynamic websites or Web applications. PHP stands for Hypertext Pre-processor. It is very easy to learn and quite cost-efficient. It also supports OOP structures.

C

C is a general-purpose, procedural computer programming language supporting structured programming, lexical variable scope, and recursion, with a static type system. C language is the base language of all advanced computer languages.

C++

C++ is an object-oriented, generic and functional programming language with low-level memory manipulation. It is an extension of C language. C++ offers the feature of portability or platform independence. It is largely machine-dependent.

C#

C# is a general-purpose, multi-paradigm programming language encompassing strong typing, lexically scoped, imperative, declarative, functional, generic, object-oriented, and component-oriented programming disciplines. It is used to build dynamic games.

Java

Java is a general-purpose programming language that is class-based, object-oriented, and designed to have few implementation dependencies. Java is used to create applications that run on a single computer or be distributed among servers/ clients in a network.

Swift

Swift is a general-purpose, multi-paradigm, object-oriented, functional, imperative and block structured language. Swift is fast, safe and expressive. Swift is the future of Apple/iOS development.Swift is a powerful and intuitive programming language.
